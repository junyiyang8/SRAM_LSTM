Steps:
1. download "Google Speech Commands Dataset"  from https://www.kaggle.com/datasets/neehakurelli/google-speech-commands
2. preprocessing data: run this file "prepare_data.py" under folder 'data_utils'. then put data after preprocessing into the folder'SRAM_LSTM'.
3. run "main_for_training.py" for training.
4. you can directly run 'main_for_test.py' for getting test accuracy results.
